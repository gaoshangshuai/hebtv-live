## 🚀 使用方法

### 在APTV中播放：
1. 下载APTV应用（App Store或TestFlight）
2. 打开APTV，点击右下角设置
3. 选择"远程文件"
4. 输入上面的M3U8地址
5. 返回即可观看

### 在其他播放器中使用：
- **VLC**: 媒体 → 打开网络串流 → 粘贴M3U8地址
- **IINA** (Mac): 文件 → 打开URL → 粘贴M3U8地址
- **PotPlayer** (Windows): 右键 → 打开 → 打开链接
